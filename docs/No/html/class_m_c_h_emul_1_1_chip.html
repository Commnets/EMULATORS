Detailed Description

A chip is a specialized element within the computer (different that the CPU).
All chips are set with the full memory accesibl when the computer is initialized, unless something specific is said initializing the chip itself!.

To initialize the chip, when "the power is set up".

Returns true, when verything was ok, and false in any other circusntance.

To simulate th behaviour of the chip. It has to be defined per chip.
Returns true if everything was ok, and false in any other circunstance.
The last error could be recovered from the variable _lastError in that case.

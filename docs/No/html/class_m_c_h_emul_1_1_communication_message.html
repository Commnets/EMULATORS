The very important method.
This method execute what was required from the computer.
It must be overloaded per any specific behaviour.
The communication message can have an answer, and it is given in the second parameter. The method returns a number than can be additional info for the invoker. NOTE that Default result should be 0 meaning "No additional info for invoker".

Detailed Description

A builder class to create the messages.
An instance of this class is created in the Computer.
The structure of the message should be as follow:
1 byte (char) indicating the code of the message.
1 byte (char) indicating the number of parameters after.
a string indicating the ip (and port) origin of the message. X bytes with the parameters of the message. Usually that parameters are in the form NAMEATTR=VALUE.
That parameters should be separated using commas.
Neither names nor values can have neither equal nor comma sign in the middle.

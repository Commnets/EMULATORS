Detailed Description

Now it is time to define the parser itself.
To convert a file text (or text) into an internal structure.
The parser is a line parser. That is, it is only able to parser a line.
There can be added specific instruction parsers but always a comment command parser must exist.
It is used to determine whether a line finishes or not.

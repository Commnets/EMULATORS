Detailed Description

The class IODevice represents anything connected to any Chip of the motherboard.
The ports (RS232, Serial, IEEE, etc...) are typically elements comnnected to communication chips but also the screen or the keyboard.
Connected to a device there might be different peripherials.
For some specific devices (like screen or keyboard) the peripheral connected is "implicit" and there is no other way to add other different, but it has to be opened to other possibilities!

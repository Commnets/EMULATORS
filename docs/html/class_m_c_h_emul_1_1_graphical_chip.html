Detailed Description

GraphicalChip is very connected with a Screen.
The graphical chip uses the ScreenMemory to write.
The same object will be taken by th Screen to finaly put the graphics out to the display.
The instance of the ScreenMemory object used is created in the method "createScreenMemory ()" that has to be overload.
When a graphic is ready (complete) to be sent to the screen, the method "setGraphicsReady (true)" should be invoked.
The Screen, when output the raphics to the display invokes back the method "setGraphicsReady (false)".

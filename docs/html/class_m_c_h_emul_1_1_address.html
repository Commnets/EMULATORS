Detailed Description

Representing and address that can vary from 0 to a limit. An initial value and the limit is given at construction time. If no limit is given the maximum possible in the framework is taken.

◆ fromStr()

MCHEmul::Address MCHEmul::Address::fromStr ( const std::string &  str)
static

To create an Address from an string.
The string should start with $ if written in hexadecimal, with 0 if it is in octal and with no 0 number in decimal:
e.g $D400 (hexa), 07600 (octal), 53248 (decimal).
If the str is not valid, then an empty address will be given.
